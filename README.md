Data and Processing Code for Permeability Automation System. 

Datafiles for publication are in "data" folder. In this folder there are subfolders for the equilibration regime processed data and their fits.
Raw csv files and calibration files are in the data folder directly. 

Kalman_Filtering_for_Publication.ipynb is used to determine the equilibration regime and saves data fits for all data shown in the work.
It was also used to generate publication Figures 5, 6 and 7.


Data_Processing_for_Publication.ipynb was used to process calibrations, and reads in fits generated through the Kalman_Filtering_for_Publication code.
It was used to generate Figures 2, 3, and 4.


Code used to run the instruments and scale designs for custom parts are available in the respective folders.
